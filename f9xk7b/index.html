Cara reset password:

Cara 1 — Via GitHub (paling mudah):
1. Buka repository GitHub Anda
2. Klik file admin/script.js → klik ikon pensil
3. Ganti nilai MASTER_HASH menjadi 'GANTI_DENGAN_HASH_ANDA'
4. Klik Commit changes → tunggu 1–2 menit
5. Refresh halaman ini → section setup muncul → set password baru

Cara 2 — Via Browser Console (tanpa edit kode):
1. Tekan F12 → tab Console
2. Ketik perintah ini dan tekan Enter:
generateHashFor('password_baru_anda').then(h => console.log('Hash:', h)) 3. Copy hash yang muncul di Console
4. Buka admin/script.js di GitHub → ganti MASTER_HASH → Commit
Setup Pertama Kali

Langkah 1: Masukkan password yang ingin Anda gunakan → klik Generate Hash.
Langkah 2: Copy hash yang muncul.
Langkah 3: Buka file admin/script.js di GitHub → ganti GANTI_DENGAN_HASH_ANDA dengan hash tersebut → Commit.
Langkah 4: Tunggu 1–2 menit → refresh halaman ini → login dengan password tadi.
